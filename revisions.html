Revisions To Project

I asked Shawn Cannon who is also in the program for a 2nd set of eyes on my website project. We focused in on the main page and made some significant changes. The page prior was a little messy and dull with no real styling elements. I added a background image, border for text with padding, corrected margins on page to make the page tighter and appear more clean..

Picture included below.

With her 2nd set of eyes I was able to get a better feel on what layouts look better. She thought it would look more professional if I have the pictures and text centered. This gave my site a much more aesthetically pleasing feel, and I personally loved the changes

The Changes
1. Centered the text and photo
2. Corrected margins to make page tighter
3. Corrected color scheme and added Ford Emblem background
4. Added border to around text
